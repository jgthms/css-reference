#opacity

Defines how opaque the element is.

default opacity: 1;

The element is fully opaque.

Hello world

opacity: 0;

The element is fully transparent.

Hello world

opacity: 0.3;

Any value between 0 (zero) and 1 (one) will make the element semi transparent.

Hello world
